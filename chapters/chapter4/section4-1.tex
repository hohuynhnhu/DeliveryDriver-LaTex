\section{Môi trường triển khai}

\subsection{Phần cứng}

Hệ thống được triển khai và kiểm thử trên máy tính cá nhân với cấu hình:

\begin{itemize}
    \item Hệ điều hành: Windows 11
    \item CPU: Intel Core i5
    \item RAM: 16GB
\end{itemize}

\subsection{Phần mềm và công nghệ}

Các công nghệ được sử dụng bao gồm:

\begin{itemize}
    \item Frontend Mobile: Flutter
    \item Backend: FastAPI
    \item Cơ sở dữ liệu: PostgreSQL (Supabase)
    \item Xác thực người dùng: Supabase Authentication
    \item Triển khai dịch vụ: Docker
\end{itemize}
