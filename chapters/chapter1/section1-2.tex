\section{Công nghệ Frontend}

\subsection{Flutter (Mobile App)}

Flutter (Dart, Google) cho phép xây dựng mobile app (iOS/Android) từ single codebase. Ưu điểm:

\begin{itemize}
    \item Hot reload nhanh, UI đẹp (60fps)
    \item Package ecosystem phong phú
    \item Tích hợp Google Maps, GPS tracking dễ dàng
\end{itemize}

\textbf{Packages chính sử dụng:}
\begin{itemize}
    \item \texttt{google\_maps\_flutter}: Hiển thị bản đồ và tuyến đường
    \item \texttt{geolocator}: Lấy vị trí GPS real-time
    \item \texttt{supabase\_flutter}: Kết nối database và authentication
    \item \texttt{provider}: Quản lý state
\end{itemize}

\subsection{Nuxt.js (Web)}

Nuxt.js là framework Vue.js với server-side rendering (SSR) và static site generation. Web admin được xây dựng để quản lý hệ thống, phân công đơn hàng và theo dõi tài xế.

\textbf{Công nghệ chính:}
\begin{itemize}
    \item \textbf{Vue 3}: Progressive JavaScript framework
    \item \textbf{Nuxt 3}: Meta-framework với SSR, file-based routing
    \item \textbf{Pinia}: State management (thay thế Vuex)
    \item \textbf{Tailwind CSS}: Utility-first CSS framework
\end{itemize}

\textbf{Thư viện UI và visualization:}
\begin{itemize}
    \item \texttt{Leaflet}: Thư viện maps mã nguồn mở (thay thế Google Maps)
    \item \texttt{Chart.js + vue-chartjs}: Biểu đồ và thống kê
    \item \texttt{Lucide Vue}: Icon library
    \item \texttt{vue3-toastify}: Thông báo toast
\end{itemize}

\textbf{Firebase Integration:}
\begin{itemize}
    \item \texttt{firebase}: SDK cho authentication, storage
    \item \texttt{vuefire}: Vue bindings cho Firebase
\end{itemize}
