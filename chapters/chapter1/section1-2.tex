\section{Công nghệ Frontend}

\subsection{Flutter (Mobile App)}

Flutter (Dart, Google) cho phép xây dựng mobile app (iOS/Android) từ single codebase. Ưu điểm:

\begin{itemize}
    \item Hot reload nhanh, UI đẹp (60fps)
    \item Package ecosystem phong phú
    \item Tích hợp Google Maps, GPS tracking dễ dàng
\end{itemize}

\textbf{Packages chính sử dụng:}
\begin{itemize}
  \item \textbf{google\_maps\_flutter}: Hiển thị bản đồ và tuyến đường
  \item \textbf{geolocator}: Lấy vị trí GPS real-time
  \item \textbf{supabase\_flutter}: Kết nối database và authentication
  \item \textbf{provider}: Quản lý state
\end{itemize}

\subsection{Nuxt.js (Web)}

Nuxt.js là framework Vue.js với server-side rendering (SSR) và static site generation. Web admin được xây dựng để quản lý hệ thống, phân công đơn hàng và theo dõi tài xế.

\textbf{Công nghệ chính:}
\begin{itemize}
    \item Vue 3: Progressive JavaScript framework
    \item Nuxt 3: Meta-framework với SSR, file-based routing
    \item Pinia: State management (thay thế Vuex)
    \item Tailwind CSS: Utility-first CSS framework
\end{itemize}

\textbf{Thư viện UI và visualization:}
\begin{itemize}
    \item Leaflet: Thư viện maps mã nguồn mở để hiển thị bản đồ
    \item OSRM (Open Source Routing Machine): API routing cho OpenStreetMap, tính toán tuyến đường
    \item Chart.js + vue-chartjs: Biểu đồ và thống kê
    \item Lucide Vue: Icon library
    \item vue3-toastify: Thông báo toast
\end{itemize}

\textbf{Firebase Integration:}
\begin{itemize}
    \item firebase: SDK cho authentication, storage
    \item vuefire: Vue bindings cho Firebase
\end{itemize}
